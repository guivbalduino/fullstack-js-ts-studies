# 📚 Fullstack JS/TS Studies

Repositório de estudos e exercícios cobrindo toda a stack JavaScript/TypeScript do zero ao avançado — front-end e back-end — com foco em projetos práticos, segurança e boas práticas de mercado.

---

## 🛠️ Conteúdo

**JavaScript**
- Fundamentos, lógica de programação, funções, arrays e objetos
- Programação orientada a objetos (POO) e funcional
- Async/Await, Promises, Fetch API, Ajax
- Tooling: Webpack e Babel

**Node.js & Back-end**
- Node.js, Express, MongoDB com Mongoose
- API REST com JWT, Sequelize (MySQL/MariaDB)
- NestJS e Strapi

**Front-end**
- React JS com class components e functional components
- React Hooks, Redux, Redux Saga, React Router Dom
- Next.js com SSR, SSG e ISR

**TypeScript**
- Tipos básicos e avançados, Generics, Decorators
- Classes, Interfaces, Namespaces e Modules
- Princípios SOLID e Design Patterns GOF
- Testes com Jest

**Banco de Dados**
- SQL com MySQL/MariaDB e PostgreSQL
- NoSQL com MongoDB
- Query Builder com Knex.js

**Infraestrutura**
- Deploy em Ubuntu Server com NGINX, PM2 e Let's Encrypt
- Configuração de SSH, HTTPS e segurança do servidor
- GitHub Webhooks para Continuous Deployment

**Extras**
- HTML5 e CSS3
- Expressões Regulares (Regex)
- Comandos Linux/Unix
